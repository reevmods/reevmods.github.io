Notepage, The Simple Note Editor Tool.


Download here!

Simple, powerful and functionally designed.



Completely Free!

Notepage was programmed with the idea to be free forever. We use free and unlicensed technologies and want to create a good program at no extra cost.

Simply Designed

The Notepage design is designed to be easy for anyone to use. We have renounced unnecessary feature to keep it as simple as possible.

Designed for everyday use

Notepage is Free and Very Simple. But that does not mean that it spares on features! We add new features with updates and have everything you need in everyday life already inserted.


Designed to provide a great experience.




From Start to Finish, Simpel!

Notepage is so easy to use that you will not have a single problem. Should that be the case, our support and our Github will be happy to help.

Huge performance using few resources!

Due to the lightweight construction of Notepage it is extremely fast and reacts almost without delay. Since it is based on Electron and therefore uses Javascript, it is also resource-saving and therefore suitable for every PC.

Safe saving!

We know it happens to everyone at one point, the PC fails, power failure or you forget to save. Notepage offers a solution! Notpage saves automatically when you quit the software and has a button for all who want to make sure again that it is saved


This ensures that the note actually is saved.
